
---------------------  ЦДА ----------------------
ввод: xn, yn, xk, yk

анализ на вырожденность. если точка - выход
dx = xk - xn
dy = yk - yn

l = max ( |dx| , |dy|)
sx = dx / l
sy = dy / l

x = xn
y = yn

ЦИКЛ (по i от 0 до l + 1) {
    высветить пиксель (round(x), round(y))
    x += sx
    y += sy
}
